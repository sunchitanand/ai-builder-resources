# ChatGPT Work — The Full Setup Guide

Based on [OpenAI's announcement](https://x.com/OpenAI/status/2075274271845404744) and the [official blog](https://openai.com/index/chatgpt-for-your-most-ambitious-work/) — July 9, 2026.

---

## What Changed

ChatGPT is no longer just a chatbot. **ChatGPT Work** is a new agent — powered by GPT-5.6 and Codex — that:

- **Stays with a project for hours** autonomously (not minutes)
- **Connects to your apps** — Slack, Teams, Google Drive, SharePoint, email, calendars, CRMs
- **Creates finished materials** — docs, slides, spreadsheets, and web apps ("Sites")
- **Runs scheduled tasks overnight** while you're away
- **Uses your desktop** — Computer Use clicks through apps, types, moves files

The key shift: you describe an outcome, and it does the entire task — not just answers a question.

## The Two Ways to Use It

### 1. Live, interactive work
Give it a goal, watch it work, answer questions, approve important actions. Best for tasks you already know well — a budget variance, a campaign brief, sales-meeting prep.

### 2. Scheduled Tasks (the overnight workhorse)
Ask it to run on a schedule or when an event happens:
- Review new Slack updates each week → refresh a meeting agenda
- Check dashboards each morning → send a summary before you wake up
- Monitor customer feedback → turn recurring themes into product ideas

---

## Quick Start (Day One)

1. **Connect your top 3 apps first** — Slack, Drive, email. ChatGPT auto-detects when to use them (or type `@` + app name to force it).
2. **Set up one Scheduled Task** that runs every morning — a dashboard summary or Slack digest is the easiest first win.
3. **Try Computer Use on desktop** for one repetitive workflow — file organization or form filling.
4. **Start with tasks you already know well** — you'll immediately see where it saves hours vs. minutes.
